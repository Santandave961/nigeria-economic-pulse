## Overview
Nigeria Economic Pulse is a comprehensive data analysis dashboard covering 23 years of Nigerian economic data (2000–2023). Built with Python, Pandas and Plotly, it provides interactive visualizations of GDP, inflation, exchange rates, oil dependency, and unemployment trends — with data-driven insights for each indicator.

---

##  Features
-  **GDP Analysis** — Total GDP and year-on-year growth rate (2000-2023)
-  **Inflation Tracking** — Rate vs CBN target with high-inflation years highlighted
-  **Exchange Rate History** — USD/NGN depreciation with key event markers
-  **Oil vs Non-Oil Revenue** — Diversification progress over 23 years
- **Unemployment Trends** — Labour market analysis
-  **Correlation Matrix** — Relationships between economic indicators
-  **Year Range Filter** — Analyze any custom time period
-  **CSV Export** — Download the full dataset

---

## Key Insights
- Nigeria's GDP peaked at **$568B** in 2014 before the oil price crash
- The Naira depreciated **782%** from ₦102 in 2000 to ₦900 in 2023
- Oil revenue share fell from **83%** in 2000 to **38%** in 2023
- Inflation hit **24.5%** in 2023 — driven by fuel subsidy removal and naira float
- Unemployment rose from **3.8%** in 2000 to **40%** peak in 2017

## Installation

```bash

# Install dependencies

# Run the dashboard
streamlit run app.py
```
